# Changelog

## [2.0.1] - 2026-02-23
### Changed
- Removed hard dependency on `ThingWorx_Analytics_Builder_Extension;9.7.1` in the PTCDTS_TWXLLM project XML.
- The accelerator can now be imported into ThingWorx 9.7+ environments both with and without ThingWorx Analytics Builder installed.

## [2.0.0] - 2025-10-25
### Added
- Initial release of the ThingWorx LLM Accelerator v2, including:
  - PTCDTS.TWXLLM project
  - DEMO_BEANPRO_DATA.csv & DEMO_BEANPRO_METADATA.json
  - Using LLMs with ThingWorx V2.pdf guide
